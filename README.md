# Estudar IA - WIP!

## Sobre o projeto

Este projeto é um trabalho em andamento. O repositório é um monorepo utilizando nx.dev, com 2 aplicações:
- **estudar-ia-web**: Front-end do projeto, aplicação web desenvolvida com TanStack Start e React.
- **ai-tutor-service**: Serviço Python com agentes de IA utilizados no projeto.

Para mais informações sobre cada uma das aplicações, consulte os respectivos READMEs.
